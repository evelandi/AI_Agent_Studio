# Contributing

Thanks for contributing to AI_Agent_Studio.

## Development Setup

1. Use Python 3.14+.
2. Run the app with:

```bash
/opt/homebrew/bin/python3 main.py
```

## Pull Requests

1. Create a feature branch from `main`.
2. Keep changes focused and small.
3. Ensure `main.py` compiles before pushing:

```bash
/opt/homebrew/bin/python3 -m py_compile main.py
```

4. Open a PR with a clear description of the change.
