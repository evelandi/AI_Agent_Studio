# AI_Agent_Studio

Minimal Hello World Python workspace.

## CI

GitHub Actions validates `main.py` on every push and pull request to `main`.

## Run

```bash
/opt/homebrew/bin/python3 main.py
```

## VS Code Task

Use the `Run Hello World` task to execute the script from VS Code.

## Debug

Start the `Python: main.py` debug configuration from VS Code.
